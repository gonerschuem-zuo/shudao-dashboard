# 📊 书道心力模型 · 教学数据看板

## 🎯 简介

这是一个面向书法培训机构的教学数据可视化看板，基于「书道心力模型」理论设计，帮助老师跟踪学员的学习进度和成长趋势。

---

## 📈 功能特性

### 1. 统计概览
- 重点关注学员数量
- 高潜学员数量
- 进步中学员数量
- 班级总人数

### 2. 数据可视化
- 学员阶段分布饼图
- 技法掌握热力图
- 心法觉醒分布图
- 风险预警雷达图

### 3. 进步趋势追踪
- 单人进步趋势折线图
- 双人对比分析图

### 4. 学员档案管理
- 71名学员详细档案
- 按类型筛选（需关注/高潜/进步中/积累中）
